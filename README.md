# Yugr19-Tfmon
Tfmon is an OpenWrt plugin that uses nf_conntrack to provide accurate real-time network speed statistics.

本插件旨在解决 MT798X、IPQ60XX、IPQ53XX、IPQ50XX、IPQ807X（Cortex‑A53）等低性能处理器平台在开启硬件加速后，bandix 等插件无法准确反映设备实时网络速率的问题。Tfmon 基于 kmod‑nf‑conntrack 实现，是一款具备高效率与高兼容性的网络速率统计工具。
